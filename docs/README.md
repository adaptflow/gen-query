# GenQuery Documentation Site

This directory contains the Docusaurus documentation site for GenQuery.

## Install dependencies

```bash terminal
npm install
```

## Start local development server

```bash terminal
npm run start
```

## Build production site

```bash terminal
npm run build
```

## Serve production build locally

```bash terminal
npm run serve
```

The generated production output is written to `website/build/` and is ignored by Git.
